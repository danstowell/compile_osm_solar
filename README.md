# compile_osm_solar
Script to parse an OSM XML extract for solar PV data

Requirements:

* Python 3
* Python packages:
   * numpy
   * pandas
   * matplotlib
   * scipy
* [osmium](https://osmcode.org/osmium-tool/)

I use data extract downloads [from Geofabrik](http://download.geofabrik.de/europe/great-britain.html).
